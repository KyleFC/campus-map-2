Recommended Software Downloads

We have created a list of some great software programs that we highly recommend for your personal computer. Most of them are either very inexpensive or completely free.

We do not offer support for these programs, but most of the manufacturers do have support resources on their websites. Take a look and let us know if you have questions.

‌
Web Browsers & Online Tools
Google Chrome
Google's free Web Browser
Mozilla Firefox
Another free web browser from Mozilla
Java Runtime Environment
Java's multimedia active web content plug-in
Respondus LockDown Browser
Exam tool for use with Blackboard
‌
Office Software
Adobe Reader
Adobe's PDF viewing software
Foxit PDF Reader
Another free PDF reader. Smaller, simpler, and better performance than Adobe Acrobat Reader
Primo PDF Reader
A free tool to create PDF files
Free PDF Online or PDF Converter
A website that can convert documents to PDF for you
Free PDF to Word or HTML conversion
A website that can convert PDF files to editable word or HTML files
OpenOffice.org
A free alternative to Microsoft Office. This suite will allow you to open and edit Microsoft Office documents.
Microsoft Office 365
Microsoft Office 365 is available for all active students through their @eagles.cui.edu emails. Sign up on this site and set up an Office account to gain access. You will likely be asked to go through two-factor authentication. You can find assistance via this guide: Microsoft Office 2-Factor Authentication.
‌
Miscellaneous
7-zip File Manager
A free tool for opening, editing, and creating "zip" compressed archive files. Very small, fast, and powerful.
VLC (Video Lan Client)
An open-source and free multimedia playback tool. Fast and simple to use.
ImgBurn
A free tool for burning data and iso files to CD, DVD, or Blu-ray discs.
AxCrypt
A free tool for encrypting and password protecting sensitive files on your computer
TrueCrypt
Another free tool for encrypting and password protecting sensitive files on your computer
