WEBSITE REQUESTS
Marketing

We provide solutions for building and maintaining web pages that can help you get your content online. Below is a round-up of Web services we offer for cui.edu.

EXPLORE ▾
Submit a Ticket

Tickets are updates and additions to the website that require 1-3 business days to complete. Requests that require more than 1-3 days to complete will be considered as a project.

Ticket requests include:

Content Updates (e.g., outdated information, inaccuracies)
Removing a Faculty/Staff Member Bio
Adding an Article
Blog Post
Podcast

SUBMIT A TICKET

Submit an Event

We hope to expedite the process of getting your event added to the event calendar and promoted through the website. This form will walk you through the information we need to publicize your event.

Student events for the calendar on MyCUI can also be submitted this way.
Events encompass anything with a specific date(s), start time, and location (on-campus, off-campus, or online).

SUBMIT AN EVENT

Submit a Request for a Web Project

Projects typically require more than 1-3 business days to complete. Projects require an approval and scheduling process. You will receive an email from the project manager with scheduling details and any clarifying questions.

These requests include:

Microsites (a welcome / homepage with 1+ subpages)
Blogs or Podcasts
A new webpage (including new event pages)
Program branding and messaging updates
A new form or updates to an existing form
Content updates/additions/removals that are more complex than misspellings, outdated information, or inaccuracies

SUBMIT A PROJECT REQUEST

Submit a Faculty Bio

Adjunct and full-time professors can add a new profile or update an existing one.

SUBMIT A FACULTY PROFILE
